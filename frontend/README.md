# Frontend - Restaurant Analytics

React + Vite + TypeScript frontend.

## Setup Local

```bash
# Instalar dependências
npm install

# Rodar dev server
npm run dev

# Build production
npm run build
```

## Estrutura

```
src/
├── components/   # Componentes React
├── hooks/        # Custom hooks
├── lib/          # Utilitários
├── store/        # State management (Zustand)
└── types/        # TypeScript types
```

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- TanStack Query
- Recharts
